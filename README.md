## 🎯 What Is This?

A **single HTML file** that contains a complete, interactive Python course — from your very first `print("Hello")` to building real AI-powered backends with FastAPI and Claude API.

No frameworks. No dependencies. No internet required after download. Just open `index.html` in your browser and start learning.

> **Built by [Ranjan Kumar Jena](https://github.com/rjena3214)** — an AI developer on a 75-day journey from beginner to professional. This course is everything I wish I had when I started.

## 🚀 Quick Start

**Option 1 — Use online (recommended)**
```
https://rjena3214.github.io/python-course
```

**Option 2 — Download and use offline**
```bash
git clone https://github.com/rjena3214/python-course.git
cd python-course
start index.html    # Windows
open index.html     # Mac/Linux
```

**Option 3 — Single file download**

Click `index.html` → `Raw` → `Ctrl+S` to save → open in any browser.
